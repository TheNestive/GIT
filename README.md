# Nova Atelier

Minimal Next.js starter for rebuilding the site step by step.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- ESLint

## Commands

```bash
npm install
npm run dev
npm run build
npm run start
```

## Structure

```text
app/
  globals.css
  icon.svg
  layout.tsx
  page.tsx
components/
  layout/
    Footer.tsx
    Navbar.tsx
  ui/
    Button.tsx
```

## Notes

- Only the `/` route is kept.
- Global styles and project configuration remain in place.
- The homepage contains a simple navbar, hero, and footer.
