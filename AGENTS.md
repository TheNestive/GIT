# AGENTS.md

## Project goal
Build a premium, modern, production-ready corporate website for a future company.

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion

## Rules
- Keep code clean, scalable, and production-oriented.
- Prefer reusable components.
- Maintain a premium visual style.
- Avoid unnecessary libraries.
- Keep deployment simple for Node.js hosting.
- Do not add mock backend complexity unless necessary.
- Use semantic HTML and responsive design.
- Optimize for GitHub push and future Hostinger deployment.

## Commands
- Install: npm install
- Dev: npm run dev
- Build: npm run build
- Start: npm run start
- Lint: npm run lint

## Done definition
A task is complete only if:
1. the code compiles,
2. the app runs locally,
3. the build passes,
4. the structure is clean,
5. the result is visually polished,
6. the README is updated if needed.
